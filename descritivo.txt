🐶 TOBBY - DOCUMENTAÇÃO COMPLETA DO PROJETO
Versão 5.1 - Com Criptografia AES-256 e IA Groq

📌 1. VISÃO GERAL DO PROJETO
Tobby é um organizador financeiro pessoal com personalidade, projetado para ser um "companheiro financeiro". O nome é uma homenagem ao cachorro Tobby Maguire.

Filosofia
- Não é um app bancário (não faz transferências, pagamentos reais)
- É um organizador e acompanhante financeiro
- Foco em personalidade e conexão emocional com o usuário
- IA que conversa de forma natural e personalizada
- Privacidade em primeiro lugar (dados criptografados)

🔐 5. CRIPTOGRAFIA
- Algoritmo: AES-256-CBC
- Chave: 32 bytes (64 caracteres hex) via ENCRYPTION_KEY
- Campos criptografados: salary_encrypted, value_encrypted, purchase_price_encrypted, current_price_encrypted, total_principal_encrypted, outstanding_balance_encrypted, monthly_payment_encrypted, estimated_value_encrypted

Gerar chave: node -e "console.log(crypto.randomBytes(32).toString('hex'))"

🐛 11. PROBLEMAS CONHECIDOS E SOLUÇÕES
- Supabase desconectado: Remover /rest/v1/ da URL
- Erro PGRST125: Verificar nome da tabela (users, não tobby_users)
- Timeout no Supabase: Projeto gratuito hiberna
- Criptografia não funciona: Adicionar ENCRYPTION_KEY no Render
- Erro 401 no admin: Usar tobbyadmin2024 no header x-admin-key
- IA não responde: Configurar GROQ_API_KEY no Render

🎯 14. RESUMO PARA IA
Tobby é um organizador financeiro pessoal com personalidade (mascote cachorro). Backend Node.js com Express e Supabase (PostgreSQL). Frontend PWA puro (HTML/CSS/JS). Autenticação JWT. Criptografia AES-256 para dados sensíveis. Funcionalidades: controle de contas, investimentos, financiamentos (simulação de amortização), patrimônio (bens), leitura de nota fiscal por foto, leitura de holerite, importação de extrato (CSV/OFX/texto), informe de IR, chat com IA Groq (gratuito, modelo Llama 3.3 70B). Design responsivo com tema claro/escuro. Hospedado no Render (backend) e GitHub Pages (frontend). Variáveis obrigatórias: SUPABASE_URL, SUPABASE_SECRET_KEY, JWT_SECRET, ADMIN_KEY. Opcionais: ENCRYPTION_KEY (criptografia), GROQ_API_KEY (IA).
