📌 1. VISÃO GERAL DO PROJETO
Tobby é um organizador financeiro pessoal com personalidade, projetado para ser um "companheiro financeiro". O nome é uma homenagem ao cachorro Tobby Maguire.

Filosofia
✅ Não é um app bancário (não faz transferências, pagamentos reais)

✅ É um organizador e acompanhante financeiro

✅ Foco em personalidade e conexão emocional com o usuário

✅ IA que conversa de forma natural e personalizada

✅ Privacidade em primeiro lugar (dados criptografados AES-256-GCM)

✅ Gamificação e conquistas para engajamento

✅ Análise financeira profunda com inteligência artificial

✅ Categorias customizáveis pelo usuário

✅ Radar Financeiro com alertas de mercado

✅ Simulador de Sonhos para planejamento de metas

✅ Saúde Financeira Familiar para compartilhar objetivos

✅ Diário Financeiro com análise emocional

✅ Linha do Tempo da Vida

✅ Planejador de Aposentadoria

✅ Missões Inteligentes com IA

✅ Tobby Proativo (Briefing Matinal)

🔐 5. CRIPTOGRAFIA
Algoritmo: AES-256-GCM (mais seguro que CBC)

Chave: 32 bytes (64 caracteres hex) via ENCRYPTION_KEY

IV: 12 bytes (recomendado para GCM)

Autenticação: GCM inclui auth tag para detectar adulteração

Fallback: Compatível com dados em texto plano

Service Layer: Centralizado em encryptionService.js

Campos criptografados:
salary_encrypted (users)

value_encrypted (bills)

purchase_price_encrypted, current_price_encrypted (investments)

total_principal_encrypted, outstanding_balance_encrypted, monthly_payment_encrypted (loans)

estimated_value_encrypted, acquisition_value_encrypted (assets)

Gerar chave:
bash
node -e "console.log(crypto.randomBytes(32).toString('hex'))"

🐛 11. PROBLEMAS CONHECIDOS E SOLUÇÕES
Problema	Solução
Supabase desconectado	Remover /rest/v1/ da URL
Erro PGRST125	Verificar nome da tabela (users, não tobby_users)
Timeout no Supabase	Projeto gratuito hiberna
Criptografia não funciona	Adicionar ENCRYPTION_KEY no Render
Erro 401 no admin	Usar ADMIN_KEY correta no header x-admin-key
IA não responde	Configurar GROQ_API_KEY no Render
Erro 500 ao criar conta	Verificar coluna value_encrypted na tabela bills
window is not defined	✅ CORRIGIDO: frontend separado do backend
CORS Anthropic/Groq	✅ CORRIGIDO: chat usa backend como proxy
Service Worker 404	✅ CORRIGIDO: registrar em /Tobby/sw.js
